# Duendaso Chivaso

Juego de plataformas HTML5 en canvas. Todo en un solo archivo, sin dependencias.

## Como jugar

Abrir `duendaso_chivaso 2.html` en el navegador (o en GitHub Pages).

### Controles

- **Mitad izquierda de la pantalla** (o flecha izquierda / A): mover a la izquierda
- **Mitad derecha de la pantalla** (o flecha derecha / D): mover a la derecha
- **Boton SALTAR / Espacio / Flecha arriba**: saltar (doble salto disponible)

### Items

| Item | Puntos | Efecto |
|------|--------|--------|
| Cigarro | +100 | Controles invertidos por 7 seg |
| Sahumerio | +150 | Pantalla arcoiris + efecto epileptico + vuelta 360 |
| Moneda Chivaso | +10 | Coleccionable |
| Valen Torres | +500 | Modo Valen Torres con sonido especial |

## Archivos

Los PNG originales y el audio estan incluidos para referencia y futuras ediciones.
