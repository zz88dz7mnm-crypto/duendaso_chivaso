# Duendaso Chivaso

Juego HTML5 en canvas.

## Como jugar

Abrir `duendaso_chivaso 2.html` en el navegador.

- Mitad izquierda: mover a la izquierda
- Mitad derecha: mover a la derecha
- Boton Saltar / Espacio / Flecha arriba: salto
- Se puede hacer doble salto

## Archivos

El juego principal esta en `duendaso_chivaso 2.html`. Los archivos PNG y audio quedan incluidos para referencia y para futuras ediciones.
